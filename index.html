Section 1 · Pre-Training

Learning to Predict

At its core, an LLM is a next-word predictor. Given any piece of text, it assigns a probability distribution over every word in its vocabulary. Trained on billions of web pages, books, and code, the model learns the statistical structure of language at massive scale.

Section 2 · Vector Embeddings

Meaning as Math

Each word is mapped to a high-dimensional numeric vector called a word embedding. The model learns these representations during training so that words with related meanings end up nearby in vector space. Semantic similarity becomes geometric proximity.

Section 3 · Neural Networks

The Building Block

LLMs are a type of neural network, a computational model loosely inspired by how neurons in the brain connect and signal each other. A neural network takes some input, passes it through layers of simple mathematical operations called neurons, and produces an output.

Each neuron takes a weighted sum of its inputs, applies a nonlinear function, and passes the result forward. The network "learns" by adjusting those weights during training so its outputs get closer to the correct answers. Stack enough layers, train on enough data, and the network can learn surprisingly complex patterns.

LLMs take this idea to an extreme scale: hundreds of layers, billions of parameters (weights), and training data spanning most of the written internet. The input to each layer is a word embedding, like the ones in Section 2. The specific architecture LLMs use is called the transformer, covered in Section 4.

Section 4 · Transformers

Context is Everything

Earlier architectures processed text sequentially, word by word. Transformers process all tokens in parallel using a mechanism called self-attention: each token computes a weighted sum over every other token to gather relevant context before making a prediction.

Attention is a learned function that scores how much each word should "look at" every other word in the sequence. A high attention weight between two tokens means the model found their relationship useful for prediction. This lets the same word carry different meanings depending on context. Section 5 · RLHF

Making Models Helpful

Pre-training optimizes for text prediction, not for being useful or safe. Reinforcement Learning from Human Feedback (RLHF) is a fine-tuning stage where human raters rank model outputs, training a reward model on those preferences, and then using that reward signal to push the model toward responses that are Helpful, Harmless, and Honest.
